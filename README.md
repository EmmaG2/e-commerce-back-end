# eCommerce Backend

Este es el repositorio para el backend del eCommerce que estoy desarrollando.

## Requisitos

- [Node](https://nodejs.org/es/)

## Instalar

- Clona este repositorio
- Ejecuta el comando `npm install`
- Ejecuta el comando `npm run start`
